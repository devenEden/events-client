### API Requirements

- NODE v12+

### Getting started.

- Clone the repo from Gitlab.
- Move to the project directory `famis-client`.
- Create `.env.json` file from `.env.example.json` and update the necessary environment variables.
- Install dependecies with `npm install`.
- To run the app at production run `npm start`.
- The your app will de deployed on either port `3000`
